## Mulpuri Farm

This is an application for Mulpuri Farm built with React extended on a previous version.

## Description

React is utilized to create an application that iterates a list of produce and their attributes such as ID number, amount in stock, and price. When you click on each item it goes to a product details page featuring an image and description in addition to those properties. There is an option to go back to the home page below it.

## Getting Started

There is a list of produce and their attributes that are shown on the front-end. The logic already set up in the InventoryList and InventoryItem components that a user can add entries to the inventory.json file and they will automatically populate. The user should be sure to add any additional images to the image folder and set up the path correctly in the json.

## Installing

The `CS601_HW6_Mulpuri.zip` file contains all necessary files including HTML, CSS, JavaScript, icons, and images.

To run the application:
1. Download and unzip CS601_HW6_Mulpuri.zip
2. Open your terminal and use the cd command to navigate to the CS601_HW6_Mulpuri folder.
3. Install the project dependencies by entering the command npm install.
4. Start the server by entering the command npm start and the application will load in your browser.
