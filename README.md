## Mulpuri Farm

This is an application for Mulpuri Farm built with Create React App.

## Description

React is utilized to create an application that iterates a list of produce and their attributes such as ID number, amount in stock, and price.

## Getting Started

There is a list of produce and their attributes that are shown on the front-end. The logic already set up in the InventoryList and InventoryItem components that a user can add entries to the inventory.json file and they will automatically populate.

## Dependencies

React
HTML5
CSS
JavaScript

## Installing

The `CS601_HW5_Mulpuri.zip` file contains all necessary files including HTML, CSS, JavaScript, icons, and images.

To run the application:
1. Download and unzip CS601_HW5_Mulpuri.zip
2. Open your terminal and use the cd command to navigate to the CS601_HW5_Mulpuri folder.
3. Install the project dependencies by entering the command npm install.
4. Start the server by entering the command npm start and the application will load in your browser.
